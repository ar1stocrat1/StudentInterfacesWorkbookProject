# StudentInterfacesWorkbookProject

Готовый проект для студентов по теме **«Интерфейсы в C#»**.

## Как работать
1. Откройте `Program.cs`.
2. Запустите проект.
3. Найдите места с `TODO`.
4. Замените заглушки своими решениями.
5. После каждого шага снова запускайте проект.
